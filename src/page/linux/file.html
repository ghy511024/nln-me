文件&文本

文件管理不外乎文件或目录的创建、删除、查询、移动

  • 创建：mkdir
  • 删除：rm
  • 删除非空目录：rm -rf file目录
  • 删除日志 rm *log (等价: $find ./ -name “*log” -exec rm {} ;)
  • 移动：mv
  • 复制：cp (复制目录：cp -r )

find

基本语法 find [OPTION] [PATH] [CONDITION] [ACTION]

如果是中文的话 find [OPTION] [查找起始路径] [查找条件] [处理动作]

OPATION

-P、-L、-H：控制软连接的对待方式，用的不多。不介绍

PATH

要查找的目录路径。

  • ~表示$HOME目录
