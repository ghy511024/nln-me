打包拆包，压缩解压缩

tar

先举几个梨子


#仅打包，不压缩！
tar -cvf log.tar logxx.log 
#打包后，以 gzip 压缩
tar -zcvf log.tar.gz logxx.log
#打包后，以 bzip2 压缩
tar -jcvf log.tar.bz2 logxx.log  
#将tar包解压缩
tar -zxvf logxx.tar.gz

相关参数

  • -A 新增文件到以存在的备份文件
  • -B 设置区块大小；
  • -c 建立新的备份文件；
  • -C <目录> 这个选项用在解压缩，若要在特定目录解压缩，可以使用这个选项。
  • -d 记录文件的差别；
  • -x 从备份文件中还原文件；
  • -t 列出备份文件的内容；
  • -z 通过gzip指令处理备份文件；
  • -f <备份文件>或--file=<备份文件> 指定备份文件；
  • -v 显示指令执行过程；
  • -r 添加文件到已经压缩的文件；
  • -u 添加改变了和现有的文件到已经存在的压缩文件；
  • -j 支持bzip2解压文件；
  • -v 显示操作过程；
  • -l 文件系统边界设置；
  • -k 保留原有文件不覆盖；
  • -m 保留文件不被覆盖；
  • -w 确认压缩文件的正确性；
  • -p 用原来的文件权限还原文件；
  • -P 文件名使用绝对名称，不移除文件名称前的“/”号；
  • -N <日期格式> 或 --newer=<日期时间> 只将较指定日期更新的文件保存到备份文件里；
  • --exclude =<范本样式> 排除符合范本样式的文件。
  • df显示磁盘分区上的可使用的磁盘空间
  • tar显示磁盘分区上的可使用的磁盘空间
