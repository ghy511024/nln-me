本地开发，fis 环境搭建

前后端分离的前端项目，需要本地搭建fis 编译环境

1:安装node

node下载地址 安装过程一路双击略过

2:安装fis3

windows cmd 进入命令行


npm install -g fis3
// 安装完成后 fis3 -v 会提示安装成功
npm install -g fis3-jello
npm install -g fis-parser-node-sass // 比较难装，全看运气
npm install -g uglify-js
npm install -g fis3-postpackager-loader
npm install -g fis3-postprocessor-autoprefixer
npm install -g fis3-postpackager-uglify-packto
