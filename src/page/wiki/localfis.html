本地开发，fis 环境搭建

前后端分离的前端项目，需要本地搭建fis 编译环境

1:安装node

node下载地址 安装过程一路双击略过

2:安装fis3

windows cmd 进入命令行


npm install -g fis3
// 安装完成后 fis3 -v 会提示安装成功
npm install -g fis3-jello
npm install -g fis-parser-node-sass // 比较难装，全看运气
npm install -g uglify-js
npm install -g fis3-postpackager-loader
npm install -g fis3-postprocessor-autoprefixer
npm install -g fis3-postpackager-uglify-packto

3：编译命令脚本

  1. 脚本位置

    根目录

    bat

    cmd_release.bat

    cmd_run.bat

    src

    ...

    fis-conf.js

  2. cmd_release.bat

    fis 编译项目到，C:\Users\${用户}\AppData\Local\.fis3-tmp\ 目录下（默认的）

    
@echo off  
%~d0
cd %~dp0
cd ../
fis3 release -w -c -d ./

  3. cmd_run.bat

    启动fis本地服务器

    
@echo off  
%~d0
cd %~dp0
cd ../
fis3 server start -p 9000

  4. 剩下的就是想办法再ide 中把脚本跑起来了，netbeanse 直接鼠标右键文件，然后选择再系统中执行

    如果脚本异常，一闪而过，直接cmd 到项目目录下执行fis命令查看详细错误信息
